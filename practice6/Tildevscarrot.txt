Tilde(~) notation => 
1.Used for Approximately equivalent to version.
2.t will update you to all future patch versions, without incrementing the minor version.
 ~1.2.3 will use releases from 1.2.3 to <1.3.
3. It will update in decimals.



Caret(^) notation =>
1.Used for Compatible with version.
2.It will update you to all future minor/patch versions, 
  without incrementing the major version. ^2.3.4 will use releases from 2.3.4 to <3.0.0
3.It  will update to its latest version in numbers.  



What Is REST API?
REST (Representational state transfer) is a popular architecture that is
 used to create web services.
API Application Programming Interface is a code that allows two software 
programs to communicate with each other.
REST API is a software that allows two apps to communicate
 with one another over the internet and through numerous devices.
